  • Project

    Overview

    A mobile game is a video game played on such devices as smartphones, tablets and any portable media player. This project aims to develop a game similar to flappy bird and implement an in-game feature to enhance the experience of the game to which the original game did not have.

    Motivation

    Thanks to their portability, mobile games are a perfect way to occupy free time while commuting, waiting in lines, or during other uninteresting routines.

    Games like Flappy birds were successful because of the highly addictive gameplay and the need to accomplish a set task to beat the previous score.

    I am hoping to create a game that is simple but yet challenging at the same time. The game will entail similar mechanics to Flappy bird and will be developed into a 2d retro-style graphics. The game will feature a main character with endless city level and flying objects which will become progressively harder.

    The game aims to collect coins to purchase items to give the player an advantage or a better in-experience. These items consist of cosmetic, challengers, events, bonus levels and a character boost.

    Each season new items will be created to allow users to customise their character as they wish. Loot box will be created will cost an x amount of coins to open and after the end of each season, the items in the mystery boxes are placed in the shop so that people can purchase them directly for a higher price.

    An in-game messaging system will allow users to invite friends into their “group ladder” to compare and post their daily results either by SMS or via an in-game message. The game will free to play but will contain advertisements as a means of revenue. These ads will be shown automatically in interval making them relatively unintrusive.

    Description

    Every person these days has a mobile phone or tablet capable of playing games. Of the $109.8 billion in total revenue generated by digital games in 2018, mobile accounted for $61.3 billion. Gaming App downloads amount to around 21-25% of the total downloads. With many free mobile game development tools available, it is now easier to create a basic mobile game that is affordable without being an experienced game developer.
